memu-page json 검색기능 사용시 확인 내용
 * json으로 데이터 불러오는 기능 설정 후 검색기능 사용 *

- video-page span처럼 데이터 부분은 json이용, 불러오는 파일은 js이용하여 memu-page의 <a>데이터 불러오기
- json파일을 구성할 때 년도별 파일로 분리 (나중에 검색기능으로도 사용, json통합 스크립트 필요)

- 검색 결과를 표시할 때 <a>와 main-middle은 삭제하고 결과만 <a>형식으로 표시, 검색 삭제시 다시 원복
- 검색시 json에 있는 모든 필드 값을 쓰는것이 아니라 특정 필드 값만 표시할 수 있도록 스크립트 추가
- 한 개의 json파일이 아닌 여러개의 json파일을 통합하여 검색하는 자바스크립트 필요
- 검색시 json 탐색 과부하 방지 0.2sec 뒤에 검색결과 표시
- 검색시 json에 있는 특정 필드 값으로 자동완성 기능 구현
- 자동완성 표시시 표시갯수 제한과 css 따로 구성하여 수정이 가능하게
